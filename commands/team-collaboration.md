# Team Collaboration

Invoke the `team-collaboration` skill to run multi-agent parallel research tasks.

## When to Use
- Multiple independent analysis dimensions (stats + literature + figures in parallel)
- Multi-expert review of research protocol
- Systematic review across multiple databases simultaneously
- Handling multiple reviewer comments in parallel
- Time-critical projects needing parallel workflows

## Preset Scenarios

| Scenario | Agents | Use Case |
|----------|--------|----------|
| Parallel Literature Search | 3 Literature Specialists | Systematic review (PubMed + arXiv + Cochrane) |
| Parallel Analysis + Writing | Statistician + Writer + Figure Designer | Mid-pipeline acceleration |
| 4-Reviewer Simulation | 4 Reviewer agents + Editor Lead | Peer review simulation |
| Parallel Revision | 1 agent per Reviewer's comments | Handling Major Revision |
| Multi-Expert Protocol Review | Statistician + Methodologist + AI Expert | Study design evaluation |

## How It Works
1. Lead Agent decomposes tasks → creates Agent Team
2. Specialist agents work in parallel on independent subtasks
3. Lead monitors, resolves conflicts, ensures consistency
4. Lead merges outputs → resumes main pipeline

## Key Rule
Tasks with sequential dependencies CANNOT be parallelized. Only independent subtasks.
