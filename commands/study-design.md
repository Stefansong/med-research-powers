# Study Design

Invoke the `study-design` skill — unified entry point for all research types via built-in type router.

## Type Router (auto-selects workflow)
- **Type A** — Clinical (RCT / cohort / case-control / cross-sectional / diagnostic)
- **Type B** — Basic science (cell / animal / molecular / protein)
- **Type C** — AI/ML (imaging AI / prediction model / LLM/VLM / devices — IDEAL)
- **Type D** — Qualitative (interviews / focus groups / grounded theory)
- **Type E** — Survey (cross-sectional survey / validation)

Just describe your research — the skill identifies the type and applies the right workflow.

## Core Steps (all types)
1. Identify research type → activate corresponding workflow
2. Define participants / samples / data source
3. **Sample size / power analysis** (MANDATORY)
4. Define variables and outcome measures
5. Data collection plan → `data-collection-tools`
6. SAP outline → `data-analysis-planning`
7. Ethics considerations → `research-ethics`

## Output
`study-protocol.md` — complete research protocol.

## Hard Checkpoint
**User must confirm the protocol before proceeding.** Research type and primary outcome are LOCKED after confirmation.

## Mandatory next step
After confirmation → `journal-selection` + `data-collection-tools` + `data-analysis-planning`.
