# Journal Selection

Invoke the `journal-selection` skill to choose the best target journal.

## Workflow

### Step 1: Research profile analysis
Extract from existing artifacts: research type, innovation level, sample size, clinical translatability.

### Step 2: 5-dimension matching (1-5 per dimension)
Scope Match (×3) + Impact Match (×2) + Audience Match (×2) + Format Match (×1) + Timeline Match (×1) = 25-point score.

### Step 3: 3-tier recommendation
- **Tier 1 Reach** — aim high (≥20 points)
- **Tier 2 Target** — best fit, recommended first submission (≥16)
- **Tier 3 Safety** — high acceptance rate, fast review (≥12)

### Step 4: Extract submission specs
Word limit, abstract format, reference style, figure limits, special requirements, submission system.

## Output
`journal-selection-report.md` — ranked journals + specs + cascade strategy.

## Hard Checkpoint
**User must confirm target journal.** Format specs are locked for downstream writing.

## Mandatory next step
After confirmation → `data-analysis-planning` or `manuscript-writing` (apply journal format).
