# Submission System Guide

Invoke the `submission-system-guide` skill for help with journal submission portals.

## Workflow

### Step 1: Identify submission system
- ScholarOne / Manuscript Central → Lancet, JAMA, BMJ, Nature family
- Editorial Manager → Cell, Science, Elsevier journals
- Bench>Press → PLOS family
- OJS → selected open-source journals

### Step 2: File preparation checklist
- [ ] Main manuscript (.docx)
- [ ] Cover Letter (paste into system text box)
- [ ] Figures (TIFF/EPS, ≥300 DPI, separate files)
- [ ] Tables (embedded or separate per journal)
- [ ] Title Page (if required separately)
- [ ] Supplementary Materials
- [ ] Reporting Checklist (CONSORT/STROBE/PRISMA)
- [ ] Ethics approval certificate
- [ ] COI forms (ICMJE if required)

### Step 3: Metadata
Article type, title, running title, abstract, keywords (MeSH), author list + ORCID, funding, suggested reviewers.

### Step 4: Submit & verify
Upload → generate merged PDF → **review PDF carefully** → submit → save confirmation email + manuscript ID.

## Output
Submission checklist with system-specific notes.
