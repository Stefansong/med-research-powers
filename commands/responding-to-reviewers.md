# Responding to Reviewers

Invoke the `responding-to-reviewers` skill to write a point-by-point response letter.

## Prerequisites
- `revision-plan.md` from `revision-strategy` (recommended)
- Reviewer comments (user provides)

## Response Letter Format

For each comment:
```
## Reviewer [X], Comment [N]

**Comment:** [quote original]

**Response:** [your reply — collaborative, not defensive]

**Changes:** [specific modifications with page/line numbers]
```

## Tone Rules
- Never defensive or aggressive
- Acknowledge valid points before disagreeing
- Every disagreement backed by evidence or references
- Thank reviewers for constructive feedback

## Self-Check Before Submission
- [ ] Every comment has a response (no omissions)
- [ ] Every response points to specific changes (page/line)
- [ ] Word count still within journal limit after revisions
- [ ] New data/analyses consistent with existing content
- [ ] Re-run `pre-submission-verification` passed

## Output
Response letter (point-by-point) + revised manuscript.
