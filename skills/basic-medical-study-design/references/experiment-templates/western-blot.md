# Western Blot 实验设计模板

## 目的
验证 [蛋白X] 在 [条件A] vs [条件B] 中的表达差异

## 对照设置
- 阴性对照：未处理组 / 空载体组
- 阳性对照：已知高表达该蛋白的细胞系
- 内参：β-actin / GAPDH / Vinculin（根据目标蛋白分子量选择，避免与目标蛋白分子量重叠）

## 重复要求
- 生物学重复：≥ 3 次独立实验（独立传代的细胞）
- 上样量：每孔 [X] μg 总蛋白（Bradford/BCA 定量）

## 抗体信息（论文 Methods 必须包含）
| 项目 | 信息 |
|------|------|
| 一抗 | [公司], [货号], [克隆号], [种属], [稀释比], [孵育条件] |
| 二抗 | [公司], [货号], [稀释比] |
| 封闭液 | [类型], [浓度], [时间] |
| 检测方式 | ECL / 荧光 / 显色 |

## 定量方法
- 工具：ImageJ / Image Lab
- 方法：目标蛋白灰度值 / 内参灰度值
- 统计：配对 t 检验（同一实验内配对）或非配对 t 检验
- **必须展示**：所有生物学重复的定量数据，不只展示"最好看的那次"

## 常见问题
- 内参选择错误：如果目标蛋白 ~42kDa，不要用 β-actin (~42kDa)
- 只展示裁剪条带：越来越多期刊要求全膜图
- 过曝/欠曝：需要在线性范围内定量
