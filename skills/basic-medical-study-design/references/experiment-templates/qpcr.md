# qPCR 实验设计模板

## 目的
检测 [基因X] 在 [条件A] vs [条件B] 中的 mRNA 表达

## 对照设置
- NTC (No Template Control)：检测引物二聚体/污染
- 内参基因：GAPDH / ACTB / 18S rRNA（需先验证稳定性）
- 载体对照（如适用）

## 重复要求
- 生物学重复：≥ 3（独立实验）
- 技术重复：每个样本 2-3 个复孔
- **n 值 = 生物学重复数，不是技术重复数**

## 引物信息（论文必须写）
| 项目 | 信息 |
|------|------|
| 正向引物 | 5'-[序列]-3' |
| 反向引物 | 5'-[序列]-3' |
| 扩增片段长度 | [X] bp |
| 设计工具 | Primer-BLAST / Primer3 |
| 跨外显子 | 是/否（推荐跨外显子以排除 gDNA 污染） |

## 数据分析
- 方法：2^(-ΔΔCt)
- 前提验证：扩增效率 90-110%（标准曲线法）
- 熔解曲线：确认单一产物
- Ct 值 >35 视为不可靠

## 常见问题
- 内参不稳定：在实验条件下先验证内参表达不变
- 扩增效率差异大：目标基因和内参效率相差 >5% 时不能用 ΔΔCt
- gDNA 污染：未跨外显子的引物可能扩增基因组 DNA
